# MERN Live Chat Application

## 🚀 Live Demo
https://mern-real-time-chat-app-cqxg.onrender.com

## 📌 Features
- Real-time live messaging using Socket.io  
- Secure user authentication using JWT  
- Image sharing using Base64 and Cloudinary  
- Instant message updates without page reload  
- Responsive UI built with React  

## 🛠️ Tech Stack
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT  
- Real-time: Socket.io  
- Cloud Storage: Cloudinary
